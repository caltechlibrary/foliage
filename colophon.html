    Colophon¶

    The Foliage documentation has been written by Michael Hucka using the Sphinx document generator together with MyST-flavored Markdown. The theme is the Material theme for Sphinx, with light customizations such as the use of Google’s Atkinson Hyperlegible font. A GitHub Action takes care of creating the formatted version of the documentation and hosting it on GitHub.io at https://caltechlibrary.github.io/foliage. The formatted output can also be produced manually using commands implemented in the Makefile located in the docs/ subdirectory of the Foliage source code repository.

    The vector artwork used as a starting point for the logo and icon for Foliage was created by Alice Noir for the Noun Project. The artwork is licensed under the Creative Commons Attribution 3.0 Unported license. The vector graphics was modified by Michael Hucka to change the color.

    Unless indicated otherwise, all other artwork in this documentation was created using OmniGraffle Pro on a macOS computer. SVG versions of the diagrams were produced with the help of svg-buddy to embed fonts into the SVG files and overcome a limitation of OmniGraffle’s SVG output.
